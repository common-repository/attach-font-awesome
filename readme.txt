== Description ==
This plugin add font awesome library to the front of wordpress site and you can put font awesome css classes anywhere in your contents. After the plugin installed , in the settings > Attach Font Awesome Options  you can set load library files from Local or CDN.

== Installation ==
Install like any other plugin, directly from your plugins page.
